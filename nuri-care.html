바로 기록할 수 있어요!

🍎 iPhone — Siri 단축어
1. 단축어 앱 열기
2. + → 새 단축어 추가
3. URL 열기 동작 선택
4. 아래 URL 중 하나 붙여넣기
5. Siri에 추가 → 문구 설정
📋 URL 복사해서 사용하세요:
🤖 Android — Google Assistant 루틴
1. Google 앱 → 우측 하단 프로필
2. 설정 → Assistant → 루틴
3. + → 새 루틴 추가
4. 시작 문구: "유축 시작" 입력
5. 작업 추가 → 앱 열기 → URL 입력

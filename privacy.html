🎙️ 3. 마이크 권한 사용 안내

누리케어는 음성 명령 기능을 위해 마이크 접근 권한을 요청합니다.

  • 음성 인식은 기기 내 브라우저 API(Web Speech API)를 통해 처리됩니다
  • 음성 데이터는 서버에 저장되거나 제3자에게 전송되지 않습니다
  • 마이크 사용은 사용자가 음성 버튼을 탭할 때만 활성화됩니다
  • 권한은 기기 설정에서 언제든지 철회 가능합니다
⚠️ Web Speech API는 운영체제 또는 브라우저 공급자(Google, Apple)의 정책을 따를 수 있습니다. 해당 공급자의 개인정보 처리 방침을 함께 확인하세요.

☁️ 4. 제3자 서비스 (Firebase) 사용 안내

가족 공유 기능 사용 시 Google Firebase(싱가포르 서버, asia-southeast1)에 기록이 저장됩니다.

  • Firebase는 Google LLC의 서비스로, Google의 보안 인프라와 동일한 수준의 데이터 보호를 적용합니다
  • Firebase와 공유되는 데이터는 가족 공유 기록에 한정됩니다
  • Firebase는 제공받은 데이터를 광고 또는 마케팅 목적으로 사용하지 않습니다
  • 공유 기능 미사용 시 모든 데이터는 기기에만 저장됩니다
Firebase 개인정보 처리 방침: firebase.google.com/support/privacy
보안 방식
• 초대 코드 방식 — 6자리 랜덤 코드로 가족 간 공유 제한
• 로컬 우선 저장 — 기본 데이터는 기기에만 저장, 동기화는 선택사항

✋ 6. 사용자 권리 및 동의 철회

사용자는 언제든지 아래 권리를 행사할 수 있습니다.

  • 열람 요청: 수집된 데이터 확인 요청 (이메일 문의)
  • 정정 요청: 잘못된 정보 수정 요청
  • 삭제 요청: 모든 데이터 삭제 요청 (이메일 또는 앱 내 초기화)
  • 동의 철회: 마이크 권한 — 기기 설정에서 언제든지 철회 가능
  • 데이터 내보내기: 설정 → 데이터 내보내기(CSV) 기능 제공
사용자의 요청에 따라 권리 행사를 거부하거나 불이익을 제공하지 않습니다.

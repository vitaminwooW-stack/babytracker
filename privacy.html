NuriCare Privacy Policy

📅 시행일: 2026년 4월 8일  |  최종 수정일: 2026년 4월 8일

📋 1. 개요

누리케어(NuriCare)는 신생아 육아 기록 앱으로, 사용자의 개인정보를 소중히 여깁니다.

✅ 누리케어는 광고를 게재하지 않으며, 개인정보를 제3자에게 판매하지 않습니다.

📦 2. 수집하는 개인정보 항목

항목수집 목적저장 위치
아기 이름, 생년월일, 성별육아 기록 개인화기기 로컬
수유·수면·기저귀·유축 기록육아 패턴 분석기기 로컬
성장 기록 (몸무게·키)성장 추이 관리기기 로컬
가족 공유 기록엄마·아빠 공동 육아Firebase
사진 (육아일기)추억 기록기기 로컬
⚠️ 회원가입·이메일·전화번호 등 직접 식별 정보는 수집하지 않습니다.

☁️ 3. Firebase 사용 안내

가족 공유 기능 사용 시 Google Firebase(싱가포르 서버)에 기록이 저장됩니다.

  • Firebase는 Google의 보안 인프라를 사용합니다
  • 초대 코드 방식으로만 가족 간 공유됩니다
  • 공유 기능 미사용 시 모든 데이터는 기기에만 저장됩니다
보안 방식
• 초대 코드 방식 — 6자리 랜덤 코드로 가족 간 공유 제한
• 로컬 우선 저장 — 기본 데이터는 기기에만 저장, 동기화는 선택

🗓️ 4. 개인정보 보관 기간

  • 기기 내 데이터: 앱 삭제 시까지
  • Firebase 공유 데이터: 삭제 요청 시 즉시 삭제
  • 별도 서버에 데이터를 보관하지 않습니다

🗑️ 5. 데이터 삭제 방법

앱 내 삭제 설정 → 데이터 관리 → 전체 초기화
이메일 삭제 요청 hmw0106@naver.com 으로 요청 시 3일 이내 처리

🔒 6. 개인정보 제3자 제공

누리케어는 다음의 경우를 제외하고 개인정보를 제3자에게 제공하지 않습니다.

  • 사용자가 직접 가족 공유를 동의한 경우
  • 법령에 의해 요구되는 경우

👶 7. 아동 개인정보 보호

본 앱은 부모가 아기의 정보를 기록하는 도구입니다. 모든 기록은 보호자가 입력합니다.

✋ 8. 사용자의 권리

  • 개인정보 열람 요청
  • 개인정보 정정·삭제 요청
  • 데이터 이동 (내보내기 기능 제공)

🔐 9. 보안 조치

  • HTTPS 통신 암호화
  • Firebase 보안 규칙 적용
  • 초대 코드 기반 접근 제한

📝 10. 개인정보처리방침 변경

본 방침이 변경될 경우 앱 내 공지 또는 이 페이지를 통해 사전 안내합니다. 중요한 변경 사항은 시행 7일 전에 고지합니다.

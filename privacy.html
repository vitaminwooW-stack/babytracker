개인정보처리방침

누리케어 (Nuri Care)

📅 시행일: 2025년 4월 1일  |  최종 수정: 2025년 4월 1일

📋 1. 개요

누리케어(이하 "앱")는 신생아 육아 기록을 돕기 위한 개인 제작 웹앱입니다.

본 방침은 앱이 수집·이용·보관하는 개인정보의 처리 방식을 안내합니다.

💡 누리케어는 상업적 목적 없이 운영되며, 수집된 정보를 광고·마케팅에 활용하지 않습니다.

📦 2. 수집하는 정보

앱 사용 중 아래 정보가 기기 내 또는 Firebase에 저장될 수 있습니다.

👶 아기 정보

  • 이름 또는 태명, 성별, 생년월일·출생 시각
  • 출생 몸무게·키, 출산 방법
  • 지역, 출산 병원, 산후조리원 (선택 입력)

📊 육아 기록

  • 수유·수면·유축·기저귀 시작/종료 시각 및 내용
  • 성장 기록 (키, 몸무게)
  • 발달 마일스톤 체크 여부
  • 예방접종 체크 여부
  • 일기 텍스트 및 이미지
  • 아빠 체크리스트 기록

🔒 앱 설정

  • PIN 번호 (기기 내에만 저장, 암호화되지 않으므로 단순 잠금용)
  • 다크 모드, 알림 설정

💾 3. 정보 저장 위치

수집된 정보는 두 곳에 저장됩니다.

기기 내부 로컬 저장소 (LocalStorage / IndexedDB)

  • 앱을 설치한 기기의 브라우저 저장소에 저장
  • 외부로 전송되지 않으며 오직 해당 기기에서만 접근 가능
  • 앱 데이터를 삭제하면 즉시 삭제됨

Firebase (선택) Google Firebase Realtime Database

  • 가족(아빠) 연동 기능 사용 시에만 활성화
  • Google LLC 서버 (싱가포르 리전)에 저장
  • Firebase 개인정보처리방침 적용: firebase.google.com/support/privacy
⚠️ Firebase 연동을 사용하지 않으면 모든 데이터는 오직 내 기기에만 저장됩니다.

🎯 4. 정보 이용 목적

  • 수유·수면·기저귀 등 육아 기록 표시 및 분석
  • 아기 성장 추이 차트 제공
  • 엄마·아빠 간 실시간 기록 공유 (Firebase 연동 시)
  • 패턴 분석 및 육아 코치 메시지 생성
  • 수면 타이밍 알림
🚫 광고, 마케팅, 제3자 판매 등의 목적으로 정보를 이용하지 않습니다.

🤝 5. 제3자 제공

아래 경우 외에는 개인정보를 외부에 제공하지 않습니다.

  • Firebase (Google LLC) — 가족 연동 기능 사용 시 Google 서버에 저장 (위 3항 참조)
  • 법적 요구 — 법령에 의한 요구가 있을 경우

⏱️ 6. 정보 보관 기간

  • 로컬 저장소 — 사용자가 직접 삭제하거나 앱 데이터를 초기화할 때까지
  • Firebase — 연동 해제 또는 데이터 삭제 요청 시까지

설정 → 데이터 관리 → 모든 데이터 초기화로 언제든지 삭제할 수 있습니다.

✅ 7. 사용자 권리

사용자는 언제든지 아래 권리를 행사할 수 있습니다.

  • 열람 — 앱 내 모든 기록 직접 확인 가능
  • 수정 — 기록 수정(✏️ 버튼) 기능 제공
  • 삭제 — 기록 삭제(🗑️ 버튼) 또는 전체 초기화 가능
  • 이동 — 설정 → 데이터 백업으로 JSON 파일 내보내기 가능

👶 8. 아동 개인정보 관련

본 앱은 신생아 육아 기록 앱으로, 수집되는 정보는 부모가 직접 입력합니다.

만 14세 미만 아동의 개인정보는 보호자(부모)가 관리하며, 앱은 해당 정보를 육아 기록 목적 외에 사용하지 않습니다.

🔐 9. 보안

  • 앱 잠금을 위한 PIN 기능 제공 (기기 내 저장)
  • Firebase 데이터는 가족 고유 코드(invite code)로만 접근 가능
  • HTTPS(SSL) 통신 사용
⚠️ PIN은 단순 잠금 목적으로 암호화되어 저장되지 않습니다. 중요한 정보는 정기적으로 백업하세요.

📝 10. 방침 변경

본 방침이 변경될 경우 앱 내 공지 또는 이 페이지를 통해 안내합니다.

중요한 변경 사항은 시행 7일 전에 사전 고지합니다.

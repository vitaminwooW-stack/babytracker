누리케어 개인정보처리방침

NuriCare Privacy Policy

📅 시행일: 2026년 4월 8일  |  최종 수정일: 2026년 4월 8일

📋 1. 개요

누리케어(이하 "앱")는 신생아 육아 기록을 돕는 서비스입니다. 본 방침은 앱이 수집하는 개인정보의 종류, 이용 목적, 보관 기간 및 사용자의 권리에 대해 안내합니다.

✅ 누리케어는 광고를 표시하지 않으며, 개인정보를 제3자에게 판매하지 않습니다.

📦 2. 수집하는 개인정보 항목

항목수집 목적저장 위치필수/선택
아기 이름앱 내 표시기기선택
아기 생년월일D+일수 계산, 성장 기록기기필수
수유 기록 (시간, 양, 방법)수유 패턴 분석기기 Firebase선택
수면 기록 (시작/종료 시간)수면 패턴 분석기기 Firebase선택
기저귀 교체 기록배변 패턴 분석기기 Firebase선택
유축 기록 (시간, ml)유축 기록 관리기기 Firebase선택
사진성장 앨범기기선택
엄마/아빠 메모육아 일기기기선택
⚠️ 아기에 관한 정보는 아동 관련 민감 정보입니다. 앱은 이 정보를 마케팅, 분석, 광고 목적으로 사용하지 않습니다.

☁️ 3. Firebase 사용 안내

누리케어는 엄마·아빠 간 실시간 데이터 동기화를 위해 Google Firebase Realtime Database를 사용합니다.

  • Firebase 서버 위치: 싱가포르(asia-southeast1)
  • Firebase는 Google LLC가 운영하며, Google의 개인정보처리방침이 적용됩니다.
  • 동기화 기능을 사용하지 않으면 Firebase에 데이터가 전송되지 않습니다.
  • Firebase에 저장된 데이터는 초대 코드로 연결된 가족만 접근 가능합니다.
✅ Firebase 보안 규칙이 적용되어 인증된 사용자만 데이터에 접근할 수 있습니다.

🗓️ 4. 개인정보 보관 기간

저장 위치보관 기간삭제 방법
기기 내 저장소 (localStorage)사용자가 삭제할 때까지앱 설정 → 데이터 삭제
또는 브라우저 데이터 삭제
Firebase (실시간 데이터베이스)사용자가 삭제 요청 시까지이메일로 삭제 요청
[email protected]

🗑️ 5. 데이터 삭제 방법

사용자는 언제든지 자신의 데이터를 삭제할 권리가 있습니다.

📱 앱 내에서 삭제하기 설정 탭 → "오늘 기록 삭제" 또는 오늘의 기록에서 개별 기록 삭제 가능
☁️ Firebase 데이터 완전 삭제 요청 이메일: [email protected] — 요청 후 7영업일 이내 처리됩니다.
✅ 앱을 삭제하면 기기에 저장된 모든 데이터가 자동으로 삭제됩니다.
Firebase에 저장된 데이터는 이메일 요청이 필요합니다.

🔒 6. 개인정보 제3자 제공

누리케어는 사용자의 동의 없이 개인정보를 제3자에게 제공하지 않습니다.

  • 법령에 의한 요구가 있는 경우
  • 사용자가 초대 코드를 통해 가족과 직접 데이터를 공유하는 경우

👶 7. 아동 개인정보 보호

  • 앱은 보호자(부모)가 직접 입력한 정보만 처리합니다.
  • 아동 정보를 마케팅, 광고, 분석 목적으로 사용하지 않습니다.
  • 만 14세 미만 아동의 정보는 법정대리인(부모)의 동의 하에 처리됩니다.

✋ 8. 사용자의 권리

  • 열람권 — 자신의 개인정보 확인 요청
  • 정정권 — 부정확한 정보 수정 요청
  • 삭제권 — 개인정보 삭제 요청 (잊혀질 권리)
  • 처리 정지권 — 개인정보 처리 중단 요청

권리 행사: [email protected]

🔐 9. 보안 조치

  • HTTPS 암호화 — 모든 데이터 전송 시 SSL/TLS 암호화 적용
  • Firebase 보안 규칙 — 인증된 사용자만 데이터 접근 가능
  • 초대 코드 방식 — 6자리 랜덤 코드로 가족 간 공유 제한
  • 로컬 우선 저장 — 기본 데이터는 기기에만 저장, 동기화는 선택

📝 10. 개인정보처리방침 변경

본 방침이 변경될 경우 앱 내 공지 또는 이 페이지를 통해 사전 안내합니다. 중요한 변경 사항은 시행 7일 전에 고지합니다.
